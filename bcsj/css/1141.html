FireFox文本自动换行
flypeng 发布时间：1970-01-18 来源：鹏鹏电脑知识网 浏览：5次

文章目录

    　　文本自动换行在IE中实现方法：
    word-wrap:break-word;
    word-break:break-all;


    　　注：在要换行的内容相应的单元格或者DIV里加入，如：
    <p style=word-wrap:break-word;></p>或<td style=word-wrap:break-word;></td>


    　　文本自动换行在FireFox中的实现方法（脚本）：
    <script type="text/javascript">
    function toBreakWord(intLen, id){
    var obj=document.getElementById(id);
    var strContent=obj.innerHTML; 
    var strTemp="";
    while(strContent.length>intLen){
    strTemp+=strContent.substr(0,intLen)+"<br>"; 
    strContent=strContent.substr(intLen,strContent.length); 
    }
    strTemp+= strContent;
    obj.innerHTML=strTemp;
    }
    </script>


    　　注：以上脚本放在</head>前面。

    　　调用时如下写法：

    　　同个页面单处调用：
    <p id="content">这里是要应用换行的内容</p><script language="javascript">toBreakWord(60, "content");</script>


    　　同个页面多处调用：
    <p id="content">这里是要应用换行的内容</p><script language="javascript">toBreakWord(60, "content");</script>
    <p id="content2">这里是要应用换行的内容</p><script language="javascript">toBreakWord(60, "content2");</script>


    　　注：把应用的JS写在</p>后面，其中60表示一行要显示多少字字符，注意多个调用时ID的相应变化，不能同一个ID名称，应用上面的方法后IE也会是按设定的字符数换行，但是IE里面支持自动换行，所以只要判断一下是否为IE，如果不是IE就不要输出上面的那段JS，如果不是就要输出。
